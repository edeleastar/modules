Intermediate
Full Stack Node Development

Assuming intermediate level programming experience, accelerate into the world of full stack web application development in Javascript, Node.js and Javascript Client Frameworks.

Introduction to Node.js
Review aspects of html5, css frameworks, & simple api access. Accelerated introduction to Javascript & Node.js. Explore the structure of a Hapi.js application, learning the the fundmentals of routing, views, and session management. Incorporate nosql models, validation and database seeding techniques. Deploy the application.
APIs + JS Client Frameworks
Explore the structure of APIs in detail. Learn the fundamentals of Test Driven Development and use this to condtruct meaningful unit tests. Evole the APIs into sets of rest end points. Redevelop the front end using a the Aurelia client framework. Deploy this application. Brielfy examin API access in Android/Java.
Intermeidate
Android Development

Assuming intermedial level Java skils, a comprehensive course in modern Andriod Application Development.

Android Development 1
Introduce the fundamentals of Android Application Development. Learn how to design and implement Layouts & Activities, incorporating them in intuitive navigation mechanisms. Explore the activity life-cycle in detail and how it incorporates Fragments. Examine simple approaches to persistence.
Android Development 2
Continue to explroe the android platform, learning how to integrate Maps and the Camera. Explore how SQLIte content providers can be constructed. Learn how application as signed, including the principles behind the signing process.
Introductory
Web Application Development

A first course in web development, introducing the tools, languages and techniques for building web sites and applications.

Web Site Dev
Introduce html & css, the fundamentals or web page construction including layout, navigation and styling. Review best practices in HTML & CSS coding. Explore web templating and modern css frameworks.
Web App Dev
Review the nature of web apps and learn how to construct an app using a web development framework. Assuming basic Java knowledge, Explore the fundamentals of Model-View-Controller, persistence of models and deployment.
Advanced
Agile Principles & Patterns

A course in Agile Software Development principles, practices and patterns in the context of modern web and mobile application development.

Agile Software Development
Stating with an accelerated introduction modern Java, use this knowledge to learn Test First Development practices. Examine the SOLID principles in depth and explore their implementation in a series of case studies.
Design Patterns
An Exploration of Classic and Enterprise Design Patterns in the context of web and android application development. Implement these patterns in a series of case studies and evolve an understanding of the trade offs associated with their usage.
